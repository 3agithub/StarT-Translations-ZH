# Star Technology Translation Project (Chinese) 星際科技中文翻譯

This project is part of the official Star Technology (StarT) translation project which aims to translate StarT to as many languages as possible. This branch of the project aims to translate StarT to the 3 Chinese variants - China (Simplified), Taiwan (Traditional) and Hong Kong (Traditional, Cantonese), and Classical Chinese.

此項目為《星際科技》官方翻譯項目之漢化分支，目標將此模組包翻譯為《我的世界》現有的三種中文分支——中國（簡體）、台灣（繁體）及香港（繁體、粵語），另外也包括文言文。

## How do I translate? 如何加入翻譯？
Please head to the Discord server linked below, join the "Translation Interested" role, and go to the #translations channel, where you will find our forum for Chinese translations. Follow the instructions there.

請加入官方Discord伺服器，加入「Translation Interested」（有興趣翻譯）身分組，然後前往#translations（#翻譯）頻道，於中文翻譯論壇根據指引開始翻譯。
